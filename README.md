# 🩻 ChestScanAI

An AI-powered app that detects pneumonia from chest X-ray images using machine learning.

## 🚀 Live Demo

Try ChestScanAI here: https://chest-scan-ai--kchopra29.replit.app/

## What It Does

ChestScanAI uses an AI model to analyze chest X-rays and predict whether an image shows signs of pneumonia. The model was trained using deep learning techniques and achieved **84% accuracy** during evaluation. This project explores how artificial intelligence can support healthcare by helping identify patterns in medical images.

## Stack

* 🐍 Python
* 🧠 TensorFlow
